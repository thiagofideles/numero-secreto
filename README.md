# 🎯 Jogo do Número Secreto

Projeto desenvolvido durante os estudos de lógica de programação e JavaScript na Alura.

O objetivo do jogo é descobrir o número secreto gerado aleatoriamente pelo sistema.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- ResponsiveVoice (síntese de voz)

---

## 🧠 Conceitos praticados

Durante o desenvolvimento deste projeto foram praticados conceitos importantes de programação, como:

- Variáveis
- Funções
- Condicionais (`if/else`)
- Operador ternário
- Arrays
- Recursividade
- Manipulação do DOM
- Funções reutilizáveis
- Controle de tentativas
- Geração de números aleatórios
- Integração com biblioteca externa

---

## 🎮 Funcionalidades

- Sorteio de número aleatório
- Validação do chute do jogador
- Dicas informando se o número é maior ou menor
- Contador de tentativas
- Reinício de jogo
- Sistema para evitar repetição de números sorteados
- Narração em voz das mensagens exibidas na tela

---

## 📚 Objetivo do projeto

Este projeto faz parte da minha trilha de aprendizado em programação, com foco inicial em lógica de programação utilizando JavaScript, como preparação para estudos futuros em Python.
