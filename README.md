Projeto desenvolvido durante os cursos de lógica de programação da Alura.

O jogo consiste em descobrir um número secreto gerado aleatoriamente pelo sistema, recebendo dicas se o valor é maior ou menor até acertar.

# 📖 Descrição do Projeto

Este projeto foi criado com foco no aprendizado de lógica de programação utilizando JavaScript.

Durante o desenvolvimento foram aplicados conceitos fundamentais de programação, manipulação do DOM e organização de código.

---

# 🎮 Funcionalidades

✅ Geração aleatória de número secreto  
✅ Validação de tentativa do usuário  
✅ Dicas informando se o número é maior ou menor  
✅ Contador de tentativas  
✅ Reinício automático do jogo  
✅ Sistema que evita repetição de números sorteados  
✅ Narração das mensagens utilizando ResponsiveVoice  

---

# 🚀 Tecnologias Utilizadas

- CSS3

- ResponsiveVoice
